Kariera

IHS Markit jest dynamicznie rozwijającą się firmą. Współpracujemy z doświadczonymi specjalistami, jak również ze studentami dopiero zaczynającymi swoją karierę.
Poszukujemy ludzi z pasją, talentem, którzy chcą rozwijać swoją wiedzę i umiejętności. Aktualnie rekrutujemy na stanowiska:

Senior Software Engineer

Od Ciebie oczekujemy:

  • 5 lat doświadczenia w rozwoju oprogramowania
  • ukończonych informatycznych studiów wyższych lub adekwatnego doświadczenia zawodowego
  • posługiwania się w sposób komunikatywny językiem angielskim
  • bardzo dobrej znajomość platformy .Net i języka C#
  • zdolności analitycznych i umiejętności rozwiązywania problemów
  • otwartości na praktyki XP takie jak TDD, pair programming, code reviews
  • rozumienia istoty zwinnego podejścia do wytwarzania oprogramowania
  • zdolności interpersonalnych, umiejętności analizy i negocjowania wymagań
  • znajomości i umiejętności stosowania różnych stylów architektonicznych systemów z naciskiem na systemy rozproszone
  • dobrej znajomości zagadnień bazodanowych, optymalizacji zapytań i narzędzi typu ORM i/lub dobrego rozumienia zagadnień związanych z systemami WWW

Software Engineer

Od Ciebie oczekujemy:

  • ukończonych informatycznych studiów wyższych lub adekwatnego doświadczenia zawodowego
  • posługiwania się w sposób komunikatywny językiem angielskim
  • bardzo dobrej znajomości platformy .Net i języka C#
  • zdolności analitycznych i umiejętności rozwiązywania problemów
  • otwartości na praktyki XP takie jak TDD, pair programming, code reviews
  • znajomości metodologii AGILE
  • zdolności interpersonalnych, umiejętności tworzenia i negocjowania wymagań
  • znajomości SQL, wiedzy z zakresu baz danych, ORM, search engines i/lub znajomości języka JavaScript i technologii webowych

Software Quality Assurance Engineer

Od Ciebie oczekujemy:

  • minimum 2 lat doświadczenia w testowaniu lub programowaniu
  • doświadczenia w tworzeniu i utrzymywaniu testów automatycznych
  • umiejetności monitorowania, wyszukiwania i obsługi błędów systemów informatycznych
  • doświadczenia w opracowywaniu dokumentacji oraz scenariuszy testowych uwzględniających przewidywane „ścieżki krytyczne”
  • znajomości testów funkcjonalnych (jednostkowych, integracyjnych) i testów regresji
  • umiejetności analizowania wyników i raportowania przebiegu prac testowych
  • umiejetności przygotowywania dokumentacji testowej
  • praktycznej znajomości zapytań SQL i relacyjnych baz danych
  • umiejetności czytania dokumentacji analityczno–projektowej
  • znajomości języka angielskiego

Web Developer

Od Ciebie oczekujemy:

  • bardzo dobrej znajomości HTML5, CSS
  • praktycznego doświadczenia z preprocesorami CSS (Sass, Less, PostCSS)
  • znajomości JavaScript, jQuery
  • doświadczenia w projektowaniu UX/UI
  • znajomości metodologii AGILE
  • zdolności interpersonalnych, umiejętności tworzenia i negocjowania wymagań
  • posługiwania się w sposób komunikatywny językiem angielskim
  • mile widziana znajomość platformy .Net i języka C#

Data Transformation Associate

Od Ciebie oczekujemy:

  • wykształcenia wyższego
  • doświadczenia w pracy z danymi
  • bardzo dobrej znajomości języka angielskiego oraz drugiego języka obcego
  • biegłości w obsłudze Excel
  • umiejętności pracy w zespole
  • logicznego myślenia
  • wysokiej komunikatywności

W zamian możemy zaoferować

  • Pracę w jednym z najbardziej nowoczesnych i komfortowych biur w Gdańsku
  • Możliwość wdrażania własnych pomysłów i rozwiązań
  • Wsparcie rozwoju zawodowego
  • Udział w konferencjach i szkoleniach
  • Naukę języków obcych z native speakerem
  • Dostęp do najciekawszych technologii informatycznych
  • Pakiet benefitów: ubezpieczenie na życie, prywatna opieka medyczna, karta Multisport
